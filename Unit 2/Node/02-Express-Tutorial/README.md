# Gets

## API Endpoint

### `GET localhost:3000/`

## Specific Item

### `GET localhost:3000/api/v1/item/:id`

## Query

### `GET localhost:3000/api/v1/query?`

# Query Parameters

#### category=...

```
Filters the list depending on the category
Catagories:
  Breakfast
  Lunch
  Shakes
```

#### dir=...

```
Sorts the list depending on the direction
Directions:
  asc: ascending order
  dec: descending order
```
